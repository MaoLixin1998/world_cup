# 世界杯 RAG MVP

单机优先的世界杯 RAG 应用。

## MVP 技术栈

- Java 21 + Spring Boot 3 主业务后端
- Python 3.12 + FastAPI RAG/数据/评测服务
- React + TypeScript + Vite 前端
- PostgreSQL + pgvector
- Docker Compose 单机部署
